# CampaignPulse

Multi-Channel Marketing Campaign Analytics Dashboard

## Overview
A unified dashboard where small businesses/marketers can track all their marketing campaigns across different platforms (email, social media, ads) in one place, with automated performance insights and recommendations.

## Features
- Multi-platform data aggregation
- Unified analytics dashboard
- Automated performance insights
- Campaign recommendations
- Cross-channel comparison

## Tech Stack
- Frontend: React
- Backend: Node.js/Express
- Database: MongoDB
- Integrations: Platform APIs
